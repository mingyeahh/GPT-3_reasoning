GPT Chatbot

This model applies OpenAI GPT-3 API to get response from the language model by user input. It takes all the past conversation in the prompt. Compared to the version in OpenAI playground, this model divides the past conversations into different batches and use GPT-3 summarisation function to summarise each batch, and pass all the batch summaries to the prompt of the AI.

This is the GPT-3.5-turbo model, also known as ChatGPT. This model takes all previous dialogue as input and subsequently generates responses. However, when the memory capacity of the model is reached, it may lose some of the input information. During a conversation, when a sentence is highlighted, it indicates that some information has been lost and that the highlighted sentence represents the first sentence that the model currently has in its memory.
